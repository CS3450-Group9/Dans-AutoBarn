How to use this site

First and foremost, this is a DEMO car rental site. We don't actually rent cars and we don't take payments (although if you really want to give some money away I'm sure we can figure something out).

There are a few ways to use this site: unauthenticated, customer, tilworker, or manager. Normally each individual would have their own account and would have various responsibilities based on their account type. To use one of these account types your self, you can view our README here
